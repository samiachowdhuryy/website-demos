Introduction

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It serves as the backbone of web pages, allowing developers to define elements such as text, images, links, and multimedia content. At the core of HTML are tags, which function as building blocks that structure and format the content displayed in web browsers.

HTML Tags

In HTML, tags are the building blocks of the document and fundamental elements used to structure and format content on web pages. They provide instructions to web browsers on how to render text, images, links, and other media elements.

They also define the structure, layout, and content of a webpage.

Blockquote <blockquote> Tag

A blockquote is an HTML element used to represent a section of text that is quoted from another source. It is typically used to display longer quotes or excerpts from external sources and is visually distinct from the rest of the content.

Blockquotes are often indented to signify that the text is being quoted.

    Semantics: The <blockquote> tag is a semantic element, meaning it gives meaning to the content inside it. It indicates that the enclosed text is a quotation from another source.
    Block-Level Element: It is a block-level element, meaning it will start on a new line and typically be indented from both the left and right sides (depending on the browser's default styling or custom CSS). This helps distinguish it from the surrounding text.
    Can Contain Other Elements: Inside a <blockquote>, you can include paragraphs, images, links, and other block-level or inline elements, just like within any other block-level elements.

HTML <sub> Tag

The <sub> tag in HTML is used to define subscript text, which is text that appears lower and smaller than the surrounding text.

This is commonly used for mathematical expressions, chemical formulas, or footnotes.

Superscript text <sup> Tag

The <sup> tag in HTML is used to define superscript text, which appears slightly above the normal text line and is typically smaller. It is commonly used for mathematical expressions, footnotes, and ordinal indicators.

Makes text easier to read when referencing notes or scientific notations. Also helps screen readers interpret text correctly.

Address <address> Tag

The <address> tag in HTML is used to define contact information for its nearest ancestor element or the entire webpage. It typically contains information like names, phone numbers, email addresses, and physical addresses. Browsers often display the content inside the <address> tag in italic by default.

Avoid using it for unrelated content, such as physical locations in articles.

Time <time> Tag

The <time> tag defines a specific time (or datetime). The datetime attribute of this element is used translate the time into a machine-readable format so that browsers can offer to add date reminders through the user's calendar, and search engines can produce smarter search results.

Audio <audio> Tag

The <audio> tag is used to embed sound content in a document, such as music or other audio streams. The <audio> tag contains one or more <source> tags with different audio sources. The browser will choose the first source it supports. The text between the <audio>and <audio> tags will only be displayed in browsers that do not support the <audio> element.

Video <video> Tag

The <video> tag is used to embed video content in a document, such as a movie clip or other video streams. The <video> tag contains one or more tags with different video sources. The browser will choose the first source it supports. The text between the <video> and </video> tags will only be displayed in browsers that do not support the <video> element.

Source <source> Tag

The <source> tag allows you to specify alternative video/audio/image files which the browser may choose from, based on browser support or viewport width. The browser will choose the first <source> it supports.

Table <table> Tab

The <table> tag defines an HTML table. An HTML table consists of one <table> element and one or more <tr>, <th>, and <td> elements.

The <tr> element defines a table row, the <th> element defines a table header, and the <td> element defines a table cell.

Table row <tr> Tag

The <tr> tag defines a row in an HTML table.

A <tr> element contains one or more <th> or <td> elements.

Table data <td> Tag

The <td> tag in HTML defines a table cell within a row (<tr> ). It stands for "table data" and is used to insert data into a table.

Table header cell <th> Tag

The <th> tag in HTML defines a header cell in a table. It is typically used within a <tr> (table row) and makes the content bold and centered by default. This helps distinguish header cells from regular data cells (<td>).

Table header section <thead> Tag

The <thead> tag in HTML defines the table header section. It groups header rows (<tr> with <th>or <td> cells) in a table for better organization and accessibility. This makes tables easier to read, style, and manage.

Table body <tbody> Tag

The <tbody> tag in HTML defines the main body of a table. It groups rows of table data () excluding the header (<thead>) and footer (<tfoot>).

This improves table organization, accessibility, and styling.

Caption <caption> Tag

The <caption> tag defines a table caption. The <caption> tag must be inserted immediately after the <table> tag.

Form <form> Tag

The <form> tag is used to create an HTML form for user input.

Input <input> Tag

The <input> tag specifies an input field where the user can enter data. The <input> element is the most important form element.

The <input> element can be displayed in several ways, depending on the type attribute.

Label <label> Tag

The <label> tag in HTML is used to define a label for form elements like text inputs, checkboxes, radio buttons, and more. It provides a clear description of what a form field is for, enhancing accessibility, usability, and form navigation.

Textarea <textarea> Tag

The <textarea> tag defines a multi-line text input control. The <textarea> element is often used in a form, to collect user inputs like comments or reviews.

Button <button> Tag

The <button> tag in HTML is used to create clickable buttons in a web page. These buttons can be used to trigger actions, such as submitting a form, running JavaScript functions, or navigating to another page.

Select <select> Tag

The <select> element is used to create a drop-down list. The <select> element is most often used in a form, to collect user input.

Output <output> Tag

The <output> tag in HTML is used to represent the result of a calculation or user action. It is commonly used in forms or interactive pages where dynamic results need to be displayed, such as displaying the total of a calculation, or showing the result of a user input.

Samp <samp> Tag

The <samp> tag is used to define sample output from a computer program. The content inside is displayed in the browser's default monospace font.

Variable <var> Tag

The <var> tag in HTML is used to represent a variable in a mathematical expression, programming context, or other scenarios where a variable or placeholder is required. It is typically used to indicate a value that can change or a placeholder for some unknown value.
